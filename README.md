# 📊 Marketing Analytics & Automation Portfolio - Mindclick Agency

Este repositorio contiene los scripts, flujos de automatización y herramientas de análisis de datos desarrollados durante mi periodo de prácticas preprofesionales en la agencia Mindclick.

## 🚀 Proyectos Destacados

### 1. Web Scraping y Extracción de Datos (Metricool)
**Problema:** Limitación para extraer datos en líneas de tiempo desde la exportación nativa en PDF.
**Solución:** Intercepción de peticiones de red (DevTools) para capturar métricas de crecimiento diario en formato JSON, permitiendo un análisis histórico preciso.
* **Recursos:** Ver archivos en `/data`.

### 2. Dashboards Interactivos con Python
**Herramientas:** Python, Plotly, Pandas.
**Descripción:** Desarrollo de paneles dinámicos en HTML que permiten el filtrado cruzado de métricas por red social y tipo de contenido, optimizando la toma de decisiones estratégicas.
* **Recursos:** Ver scripts en `/scripts` y demos en `/dashboards_html`.

### 3. Automatización de Calendarios Operativos
**Herramientas:** Excel Avanzado (Fórmulas Dinámicas).
**Descripción:** Creación de una matriz de planificación que actualiza automáticamente la estructura de días y fechas al seleccionar el mes, eliminando el 100% de la carga operativa manual previa.
